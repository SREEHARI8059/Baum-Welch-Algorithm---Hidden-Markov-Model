# Baum-Welch-Algorithm---Hidden-Markov-Model
Name: sreehari m nambiar
Register Number: TCR24CS066

Description
This project implements the Baum-Welch algorithm to train a Hidden Markov Model (HMM).

The program calculates:

Initial state distribution (pi)
Transition matrix (A)
Emission matrix (B)
Probability P(O | lambda)
Convergence graph
Requirements
Python 3
numpy
matplotlib

How to run
Run the following command:

python3 baum_welch.py

Output
Final Initial Distribution
Transition Matrix
Emission Matrix
Final Probability
Graph showing convergence (baum_welch_result.png)
